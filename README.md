# kloos Calendar
RoundCubeMail Plugin Calendar 
Version 0.1

# How to install
1 - Copy the 'kloos_calendar' folder to the 'plugins' folder.
2 - Add the plugin name to the $config ['plugins'] array located in the 'config.inc.php' file.
3 - Done.
